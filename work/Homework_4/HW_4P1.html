input_dim = 2
output_dim = 1
W = tf.Variable(initial_value=tf.random.uniform(shape=(input_dim, output_dim)))
b = tf.Variable(initial_value=tf.zeros(shape=(output_dim,)))

The forward pass function

In [7]:
def model(inputs):
    return tf.matmul(inputs, W) + b

The mean squared error loss function

In [8]:
def square_loss(targets, predictions):
    per_sample_losses = tf.square(targets - predictions)
    return tf.reduce_mean(per_sample_losses)

The training step function

In [9]:
learning_rate = 0.1

def training_step(inputs, targets):
    with tf.GradientTape() as tape:
        predictions = model(inputs)
        loss = square_loss(targets, predictions)
    grad_loss_wrt_W, grad_loss_wrt_b = tape.gradient(loss, [W, b])
    W.assign_sub(grad_loss_wrt_W * learning_rate)
    b.assign_sub(grad_loss_wrt_b * learning_rate)
    return loss

tf.ones --> tf.ones is a function in TensorFlow that is used to create a tensor of a given shape and fill it with ones. It is similar to the np.ones function in NumPy.

Syntax: tf.ones( shape, dtype=tf.float32, name=None )

where,

shape: A list or tuple representing the shape of the output tensor.

dtype: (Optional) The data type of the output tensor (default is tf.float32).

name: (Optional) name for the operation.

tf.matmul --> Calculates the matrix multiplication of given two matrices/tensors. The inner dimensions of the matrices should match.

tf.Variable --> tf.Variable is a class that represents a mutable tensor. It is used to store and update values during the training process. Unlike constant tensors, tf.Variable can be updated via its assign method. To create a variable, we need to provide some initial value such as a random tensor. Example: v = tf.Variable(initial_value=tf.random.normal(shape=(3,1)))

tf.GradientTape() --> tf.GradientTape is a TensorFlow API for automatic differentiation. It is used to compute gradients of a computation with respect to its input variables.

Activation Function --> An activation function is a mathematical function that is applied to the output of each neuron in a neural network. The activation function takes the weighted sum of the inputs to the neuron, which includes the bias term, and applies a non-linear function to it. This new output is then passed on as input to the next layer of neurons in the neural network. By using an activation function, the neural network becomes capable of learning and modeling non-linear relationships between the inputs and outputs.

Loss Function: Loss function is a mathematical function that measures the difference between the predicted output of a model and the actual target output of the training data. The loss function determines how well the model is performing on the training data. The goal of training a machine learning model is to minimize this difference (aka loss) by adjusting the parameters of the model.
